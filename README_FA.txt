Amino — نسخه نصب‌شونده روی گوشی

این پوشه آمادهٔ انتشار روی هر هاست HTTPS است. همهٔ فایل‌ها باید کنار هم و در ریشهٔ سایت قرار بگیرند.

نکته مهم:
بازکردن مستقیم index.html از داخل فایل‌های گوشی برای نصب PWA کافی نیست. برنامه باید از یک آدرس HTTPS باز شود. آدرس localhost فقط برای تست روی کامپیوتر معتبر است.

نصب روی اندروید:
1. آدرس HTTPS برنامه را با Chrome باز کن.
2. داخل برنامه به بخش «نصب و پشتیبان» برو و «نصب برنامه» را بزن.
3. اگر پنجرهٔ نصب ظاهر نشد، از منوی سه‌نقطهٔ Chrome گزینهٔ Install app یا Add to Home screen را انتخاب کن.

نصب روی آیفون و آیپد:
1. آدرس HTTPS برنامه را حتماً با Safari باز کن.
2. دکمهٔ Share (مربع با فلش رو به بالا) را بزن.
3. گزینهٔ Add to Home Screen را انتخاب و سپس Add را تأیید کن.

بعد از اولین بارگیری، Amino و برنامهٔ تمرینی بدون اینترنت نیز باز می‌شوند. اطلاعات تمرین در حافظهٔ مرورگر همان گوشی ذخیره می‌شود. برای جلوگیری از ازدست‌رفتن سوابق، مرتب از بخش «پشتیبان اطلاعات» خروجی JSON بگیر.
